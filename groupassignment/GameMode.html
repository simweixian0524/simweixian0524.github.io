There are currently 3 game modes in Hollow Knight

The Standard game mode which the main story of the game unfolds.

Unlocked after finishing the game the first time. In Steel Soul Mode The Knight does not respawn upon death and if it does happen, the game ends and the save file is reset.

Unlocked after completing the first three Pantheons in Godhome. The main purpose of this game mode is to let the player to gain access to almost all of the boss fight in the game. The player is given the fully upgraded Knight, fully upgraded Nail, most of the Charms, upgraded spells and abilities without playing through the main game.
